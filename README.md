## How To Run This Project

Clone this project url :

``` https://github.com/VsalCode/fgo24-react-medium.git ```

after entering the folder, install the dependencies:

```npm install```

To run the web you can use the command :

```npm run dev```

##  dependencies used
- Vite
- Tailwind CSS
- React Router DOM
